デジタル名札・座席札メーカー

学校の先生やイベント主催者向けの、インストール不要で使える名札・座席札作成Webアプリケーションです。
ブラウザでHTMLファイルを開くだけで、A4用紙に印刷可能な名札や座席札を簡単に作成できます。

特徴

インストール不要: HTMLファイル1つで動作するため、インターネット環境さえあればどこでも使えます。

2つの作成モード:

首かけ名札モード: A4用紙1枚に8人分配置。

三角座席札モード: A4用紙1枚に3人分配置。机の上に置ける山折りガイド付き。

印刷最適化: A4サイズ（210mm x 297mm）に完全対応。切り取り線（ミシン目ガイド）も自動描画。

一括登録機能: CSV形式（名前,ふりがな）でのリスト一括登録に対応。

デザインカスタマイズ: クラス名や役職名の表示、フォント（丸ゴシック、明朝体など）、テーマカラーの変更が可能。

使い方

起動: NameTagMaker.html をPCのブラウザ（Chrome, Edge, Safariなど）で開きます。

設定: 左側のサイドバーでモードやデザインを設定します。

共通タイトルに「〇〇学校」「〇年〇組」などを入力します。

名簿入力:

リストに直接名前を入力するか、「一括追加(CSV)」ボタンからテキストデータを貼り付けます。

入力例: 山田 太郎, やまだ たろう

印刷:

画面左下の「印刷する」ボタンをクリックします。

プリンターの設定画面で以下を確認してください。

用紙サイズ: A4

倍率: 100%（または「規定」）

ヘッダーとフッター: オフ（チェックを外す）

背景のグラフィック: オン（色を印刷する場合）

ファイル構成

NameTagMaker.html: アプリケーション本体（HTML, CSS, JavaScriptが含まれています）

動作環境

最新のモダンブラウザ（Google Chrome, Microsoft Edge, Safari, Firefox）

JavaScriptが有効になっていること
